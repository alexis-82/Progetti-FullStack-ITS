## Procedura Utilizzo

Installazione di Node.js tramite file binari

I passi sono:

1. **Installare module**:
   
```
npm install
```

2. **Avviare node**:
   
```
node app.js
```

3. **Collegarsi con il Browser**:

```
http://localhost:3000
```

4. **Lista ricette**:

```
http://localhost:3000/api/recipes
```

5. **GET-POST-PUT-DELETE** utilizzare es. Thunder Client:

```
http://localhost:3000/api/recipes/(id)
```
